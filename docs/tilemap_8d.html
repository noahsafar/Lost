Detailed Description

Tilemap system for 2D level rendering and collision detection.

This module implements a tilemap system for creating and rendering 2D game levels. It supports loading from JSON files, setting individual tiles, and rendering with a tileset. The tilemap is used for level geometry, collision detection, and visual representation of game levels.

Each tile can be:

  • -1: Empty space
    • 0: Tile (platform block)
    • 1: Spike Up
    • 2: Spike Right
    • 3: Spike Down
    • 4: Spike Left
    • 5: Carrot (collectible)
