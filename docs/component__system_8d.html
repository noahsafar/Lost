Detailed Description

Entity-Component-System architecture for the game engine.

This module implements an ECS (Entity-Component-System) architecture along with a Camera class for world-to-screen coordinate transformations. The ECS pattern allows for flexible game object composition where entities are containers for components, and systems operate on entities with specific component types.

Components include:

  • TransformComponent: Position, size, and rotation
  • TextureComponent: Rendering information
  • AnimationComponent: Sprite animation state
  • ScriptComponent: Custom behavior scripts
