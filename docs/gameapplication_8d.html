Detailed Description

Core game application structure and main loop implementation.

This module contains the GameApplication struct which manages the game's lifecycle, including initialization, scene management, input handling, update/render cycles, and cleanup. It serves as the central coordinator for all game systems.

The application uses SDL3 for window management and rendering, and implements a scene-based architecture for managing different game states.
